Deep Learning-Udacity

August 15th 2017

I was proud to implement my first, solo, neural network over the summer. I’ve been enjoying Udacity’s free online course on Deep Learning. The class is taught in Python, and the nets are implemented in TensorFlow. With intuitive lectures and with some helpful, provided, code, instructors cover NNs, CNNS, and LSTM models. To teach, they primarily use the NotMINST numeric picture dataset. There’s strong forum support and some very clever/pythonic students. The class is an intermediate difficulty, and it helps to have some prior logistic regression and classification. Having started implementation of the CNNs, I would recommend this class to anyone interested in learning more about the algorithms driving innovation in AI and computer vision.

March 17th 2017

March might be over, but the madness of sports analysis never ends! Nearly 20 million NCAA March Madness brackets were made in 2017; has yours busted yet? It seems like every year some no-name high seed upsets a lower seeded team in a marvelous display of determination and grit. Given the immense amount of sports data collected on college basketball, predicting tournament winners is a data scientist’s sweetest of dreams. Why Data Science?

April 12th 2017

I still remember when I made my first linear regression model. We we’re assigned the task of constructing a univarte least squares regression to predict age from demographical attributes. My model was 100 times slower than the equivalent SKlearn function and used an excessive amount of code and memory. Regardless, I was mesmerized by the ability to predict the future solely using prior events. Soon after, on the same problem set, I constructed my first classification model using a k-nearest neighbors algorithm. I’m too embarrassed to mention how slow it was; but plotting the data using my predicted classes and comparing accuracy to the true classes made up for the frustrating hour I spent debugging such a simple, in hindsight, model. I was hooked. From there, the world of predictive analytics was opened through visualization techniques and statistical and machine learning. I started this blog to further my data science education and get feedback on my projects.
